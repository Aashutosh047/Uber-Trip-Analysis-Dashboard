# Uber Trip Analysis Dashboard | Power BI

## Project Overview

This project presents an interactive Power BI dashboard built using real-world Uber trip data. The objective of the dashboard is to transform raw transportation data into actionable business insights through data visualization, KPI tracking, and trend analysis.

The dashboard enables users to explore trip patterns, revenue performance, location-based demand, and operational metrics through an intuitive and interactive reporting experience.

---

## Business Problem

Ride-sharing companies generate large volumes of trip data daily. Extracting meaningful insights from this data is essential for understanding customer behavior, identifying peak demand periods, optimizing operations, and improving business performance.

This dashboard was developed to answer key business questions such as:

* When does ride demand peak?
* Which locations generate the highest activity?
* What are the overall revenue and trip trends?
* How do trip distances and fares vary over time?
* Which operational metrics require attention?

---

## Dataset

The project utilizes Uber trip data containing ride-related information such as:

* Pickup Date & Time
* Drop-off Date & Time
* Pickup Location
* Drop-off Location
* Trip Distance
* Fare Amount
* Trip Duration
* Payment Information

The dataset was cleaned, transformed, and modeled within Power BI before visualization.
